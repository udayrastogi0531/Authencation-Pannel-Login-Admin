## 🚀 Quick Start

### 📋 Prerequisites

```bash
📦 Node.js (v18+)
🔥 Firebase Account
🛠️ Git
💻 Modern Browser
```

### ⚡ Installation

```bash
# 📥 Clone the repository
git clone https://github.com/yourusername/authentication-panel-login.git

# 📂 Navigate to project directory
cd authentication-panel-login

# 📦 Install dependencies
npm install

# 🔥 Configure Firebase (see configuration section)

# 🚀 Start development server
npm run dev
```

### 🎉 Your app will be running at `http://localhost:5173`

## ⚙️ Customization

### 🎨 Colors

The app uses a dark theme with emerald accents. You can customize colors in `tailwind.config.js` or by modifying the CSS classes.

### 🔧 Authentication Logic

The authentication is currently mock-based. To integrate with a real backend:

1. Replace the mock users in `AuthContext.tsx`
2. Update the login/signup functions to call your API
3. Implement proper JWT token handling

### ➕ Adding New Features

- Add new routes in `App.tsx`
- Create new components in `src/components/`
- Extend the user types in `src/types/auth.ts`
